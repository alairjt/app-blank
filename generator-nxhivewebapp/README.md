## Install

Install Yeoman (http://yeoman.io/)

```
npm install -g yo
```

Install generator

```
npm install -g generator-nxhivewebapp
```


## Usage

Example 1:

```
yo nxhivewebapp
```

Example 2:

```
yo nxhivewebapp appName
```

Example 3:
```
yo nxhivewebapp appName --install
```

Create 

Options:
    install -  Install dependencies
